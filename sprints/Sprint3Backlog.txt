User Stories from Product Backlog that we will address in this sprint:
a) As a user I want to Navigate the contents of the file system
b) As a user I want to be able to create two new directories if no errors are encountered    
c) As a user I want to know where I am within the current directory.   
d) As a user I want to be able to save my current working directory and change to a new working directory. 
e) As a user I want to be able to move to the most recently saved directory 
f) As a user I want to execute commands on a prompt.
g) As a user I want to be able to print the contents of the current directory, with a new line following each of the content.
h) As a user I want to be able to print a file 
i) As a user I want to be able to replace all of a file's content with a string 
j) As a user I want to be able to view my previous commands, one command per line. 
k) As a user I want to be able to view the description of one or more commands.
l) As a user I want to be able to see content of multiple files at once separated by 3 line breaks.
m) As a user I want to be able to print a string on the shell
n) As a user I want to able to change directories which may be relative to the current directory or may be a full path. 
o) As a user I want to be able to append a string to the file's content.  
p) As a user I want to be able to view the description of one or more commands.
